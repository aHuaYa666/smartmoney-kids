# SmartMoney Kids

SmartMoney Kids is a work-in-progress MVP built during a Devpost hackathon.  
The goal of this project is to help children learn basic financial concepts through simple, interactive features.

## Status
In progress (Hackathon project)

## Planned Features
- Age-appropriate financial education content
- Simple budgeting and saving concepts
- Interactive flows designed for kids

## Tech Stack (Planned)
- Backend: Python (FastAPI)
- Database: SQLite
- Frontend: HTML / CSS / JavaScript
